## ADDED Requirements

### Requirement: Static sandbox build workspace exposes debug construction tools
The game SHALL expose the shared debug source structures and permanent test generator from the static factory demo's world build categories so developers can assemble zero-cost verification layouts on demand.

#### Scenario: Static demo build categories include debug entries
- **WHEN** the player opens the static factory demo build workspace
- **THEN** the available world build categories include clearly labeled debug item-source entries and a permanent test generator entry

#### Scenario: Static demo startup layout remains authored and non-debug by default
- **WHEN** the player enters the static factory demo before placing any additional debug structure
- **THEN** the authored startup mining, manufacturing, power, and defense loops still rely on the normal map-authored structures instead of requiring debug tools
